# Intent-based trading on Solana?

Thank to the power of Solana and Pinocchio, we create a program that enables decentralized intent-based trading on the Solana blockchain.

## Application
1. [`program/`](./program) - Solana program written in Rust using Pinocchio framework.
2. [`tests/`](./tests) - Integration tests for the Solana program.
3. [`app`/`](./app) - Frontend application to interact with the Solana program.
4. [`solver/`](./solver) - Off-chain solver service to process intents and submit transactions.
